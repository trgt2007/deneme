## 🚀 Quick Start

1. Clone repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and fill values
4. Compile contracts: `npm run compile`
5. Deploy contracts: `npm run deploy`
6. Start bot: `npm start`
